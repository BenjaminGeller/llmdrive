# Contributing

Thanks for your interest in contributing!

## Contributor License Agreement

By submitting a pull request, you agree to the terms of our [Contributor License Agreement](https://gist.github.com/BenjaminGeller/a57b597c7b0bb17d9a2db9a83110be3d). 

CLA Assistant will guide you through signing on your first PR. The CLA grants the project maintainer the right to use, modify, and relicense your contribution. This protects future flexibility for the project, including potential commercial licensing.

## How to contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit your changes
4. Push to your fork
5. Open a pull request

## Code style

Match existing patterns in the codebase. We'll discuss any style questions in PR review.

## Questions?

Open an issue or reach out via GitHub.
